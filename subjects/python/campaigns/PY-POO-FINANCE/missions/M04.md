## Objective
Introducir la librería **Pandas** para leer el histórico de transacciones y generar reportes rápidos.

---
## Description
Hasta ahora escribimos datos. Ahora vamos a leerlos y analizarlos como un Data Engineer.
Crearás un módulo de reportes que responda preguntas de negocio.

---
## Steps
1. Instalar pandas (`pip install pandas`).
2. Crear `analytics/reporter.py`.
3. Crear función `load_data(filepath)` que retorne un DataFrame.
4. Crear función `get_monthly_summary(df)` que agrupe gastos por mes y tipo.
5. Imprimir el reporte en consola.

---
## Completion Criteria
- [ ] Se usa `pandas.read_csv`.
- [ ] Se muestra un resumen agrupado (groupby) en terminal.
