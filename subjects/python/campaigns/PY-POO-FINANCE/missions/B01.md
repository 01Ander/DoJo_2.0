## Objective
Construir un motor de simulación que tome el estado actual de las cuentas y las transacciones recurrentes para proyectar el saldo futuro a 6 meses.

---
## Description
Esta misión integra todo: POO, Pandas y Lógica Financiera.
El objetivo es responder: **"¿Con mi sueldo actual y mis gastos fijos, cuánto dinero tendré en 6 meses al llegar a Colombia?"**

---
## Steps
1. Crear script `simulation.py`.
2. Definir estado inicial (Saldo actual real o simulado).
3. Definir lista de `RecurringTransaction` (Sueldo, Renta, Netflix, Comida estimada).
4. Generar transacciones futuras para los próximos 6 meses.
5. Calcular el saldo acumulado día a día.
6. Usar Pandas para exportar un Excel (`projection.xlsx`) con las columnas:
   `Fecha | Ingreso | Gasto | Saldo Acumulado`.

---
## Domain Criteria
- [ ] El sistema proyecta saldos futuros sin modificar los datos reales actuales.
- [ ] El reporte permite ver si en algún mes futuro el saldo será negativo (alerta de quiebra).
- [ ] Código limpio, modular y testeable.
